# ECF-Grapher
This project takes an ECF rating code and shows a graph of that persons chess ECF rating over time. It was made using the wonderful Godot Game Engine, mostly on the bus.

I have yet to make the code robust so if you are using it you must not enter dumb inputs.

The web build works only on localhost, github.io, ngrok.io, trycloudflare.com, and other non-professional domains as you would otherwise get a CORS error.
